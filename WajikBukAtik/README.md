# WajikBukAtik – Panduan Edit untuk Bu Atik

## 1. Ganti Foto
- Masukkan foto baru ke folder `images/`.
- Nama file bebas, misal `hero-baru.jpg`.
- Buka `index.html`, cari baris `images/hero.jpg` dan ubah menjadi `images/hero-baru.jpg`.

## 2. Ganti Teks
- Buka `index.html`.
- Semua kalimat ada di antara tag `&lt;p&gt;`, `&lt;h1&gt;`, `&lt;h2&gt;`, `&lt;h3&gt;`, atau `&lt;strong&gt;`.
- Edit langsung, simpan (Ctrl+S).

## 3. Ganti Nomor WhatsApp
- Di `index.html`, cari `628xxxxxxxx`.
- Ganti `xxxxxxxx` dengan nomor Anda (tanpa spasi, tanpa +).
- Contoh: `6281234567890`.

## Upload ke GitHub Pages (GRATIS)
1. Buat akun GitHub → klik tombang "+" (New repository).
2. Nama repository: `WajikBukAtik` → pilih "Public" → klik "Create".
3. Upload semua file & folder (drag & drop).
4. Setelah terupload, masuk: Settings → Pages → Source: Deploy from a branch → Branch: main → Save.
5. Tunggu 1-2 menit, buka alamat: `https://username-githubanda.github.io/WajikBukAtik/`.

Selesai – selamat berjualan online!
